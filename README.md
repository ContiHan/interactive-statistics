# Statistika vizuálně

Interaktivní webová aplikace pro výuku a vizualizaci základních statistických konceptů.

Cílem projektu je poskytnout studentům a laikům intuitivní nástroj pro pochopení toho, jak změny matematických parametrů (střední hodnota, rozptyl, hladina významnosti, atd.) reálně ovlivňují pravděpodobnostní rozdělení, testování hypotéz, regresní modely či časové řady. 

## Použité technologie
- Python
- Streamlit (webový framework)
- Scipy & Numpy (výpočty)
- Matplotlib & Seaborn (vizualizace)

## Spuštění lokálně

1. Instalace závislostí:
```bash
pip install -r requirements.txt
```

2. Spuštění aplikace:
```bash
streamlit run app.py
```
